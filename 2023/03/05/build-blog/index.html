搭建一个属于自己的博客

发表于 2023-03-05
  • Yume Shoka
首页

通常来说博客就是一个网站，它可以像一张名牌一样介绍自己，也可以像笔记记录你的成长历程。你可以在上面分享各种文章，给人们答疑解惑的同时提供编写及表达能力。
本文提供了一个基于 Hexo 静态博客框架的搭建方式。

博客类型

动态博客

平时我们经常看到的博客有 CSDN，博客园等。它们都是动态博客站点，动态博客得益于后端的支持，你可以在博客站点注册用户，上传，编写文章。但是有一个很大的缺陷是无法高度自定义页面外观。
通常这些站点都是以盈利为目的，通过 会员 广告 为主要盈利手段。

静态博客

创建一个人高度自定义美观的静态博客，可能是大多数人的一个小目标。面对繁琐的步骤以及维护，让很多人望而却步。静态博客通过一定的规则将文本生成 html css javascript 静态文件，而无需后端程序便可以访问。你可以灵活的自定义以及修改这些规则，从而生成自己风格的博客站点。
