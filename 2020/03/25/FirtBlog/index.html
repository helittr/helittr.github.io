NODE.JS

NODE.JS  是一个基于 ChromeV8 开发的JavaScript 运行环境
从官网下载安装 网址 https://nodejs.org/en/
安装node.js时npm也一同安装了
    npm      是它的包管理工具 源在国外
    cnpm     也是它的包管理工具 源在淘宝上
        可以运行：npm install cnpm -g --registry=https://registry.npm.taobao.org 这一条命令来安装
        使用方法跟npm一样

HEXO

HEXO 是一个基于NODE.JS 开发的博客快速搭建工具
可以从官网下载 官网 https://hexo.io/zh-cn/
也可以运行命令 cnpm install -g hexo-cli 来安装

创建一个空文件夹并进入
使用命令 hexo init 来初始化一个博客 
使用命令 hexo new 创建一片文章
使用命令 hexo clean 清理
使用命令 hexo server 开启一个服务器

将项目部署到Github上
执行命令 cnpm install --save hexo-deployer-git
在github 新建repository(仓库)
打开博客主目录下的——congfig.yml文件 在文件末尾修改为
    type: git
    repo: （仓库地址）
    branch：master
执行命令 hexo d  将博客部署到Github仓库
